# Read-01

## What is markdown?

    Markdown is a lightweight way of writing text with simple symbols that get turned into nicely formatted stuff when displayed.

## Why do we use markdown?

    We use markdown because it's simple and easy.
    
## What symbol creates headers?
"#"

## When making text bold or italicised for emphasis, it is best practice to use which symbol?

    "*"
## How do you create a link with markdown?

    Use brackets [] and parenthesis ()

## What are three symbols you can use to create an unordered list with markdown?

    You can use -, * or +
