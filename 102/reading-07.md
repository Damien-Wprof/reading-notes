# Reading 7: Programming with JavaScript

1. What is control flow?

    Control flow is the order in which the computer reads statements made in a JavaScript file.

2. What is a JavaScript function?

    Functions are reusable chunks of code made for specific tasks. 

    functions are executed when called/invoked.
    
    Functions are fundamental in all programming languages

3. What does it mean to invoke - or call - a function?

    Calling a function is like starting a car, you call it and it runs whatever/wherever it needs to run. 

4. What are the parenthesis () for when you define a function?

    The empty parenthesis are for parameters.
    Parameters allow you to send values to a function, essentially extra data.
