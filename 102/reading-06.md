# Dynamic web pages with JavaScript

1. What are variables in JavaScript?
    Variables are lines of code that require some kind of input/output and will display or interact with different outcomes based on what kind of input/output they receive.

2. What does it mean to declare a variable?
    Declaring a variable is just setting the variable.

3. What is an “assignment” operator, and what does it do?
    An assignment operator gives a variable value.

4. What is information received from the user called?
    Input.
