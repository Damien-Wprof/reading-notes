## CSS Layout

1. What is meant by “normal flow”?

    Standard placement rules.

2. What are a few differences between block-level and inline elements?

    Block-level elements:
        Take up one line of code and push others below it down
        Take up the full width of it's parent element
        Need width & height set manually

    Inline elements:
        Usually stay in the same line with surrounding content
        Take up as much width as their content needs
        Generally ignore width and height

3. ___ positioning is the default for every html element.

    Static

4. Name a few advantages to using absolute positioning on an element.

    Absolute positioning offers more features like pop-up information boxes, control menus and rollover panels

5. What is a key difference between fixed positioning and absolute positioning?

    Absolute positioning places an element relative to it's nearest positioned ancestor, fixed positioning places an element relative to the viewport.

### Learn JS

1. Describe the difference between a function declaration and a function invocation.

    Declaring a function is just creating said function whereas
    invoking a function means it has to have been defined prior.

2. What is the difference between a parameter and an argument?

    A parameter is what you use when creating a function, specifically inside the `()`.
    An argument is what you place inside the `()` when invoking the function
