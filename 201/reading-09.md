3. Describe the event object. Why is the target within the event object useful?

The event object contains information about an event that occurred, and its target property is useful because it identifies the exact element that triggered the event, allowing precise event handling.

4. What is the difference between event bubbling and event capturing?

Event capturing handles an event from the outermost parent down to the target element, while event bubbling handles the event from the target element back up through its parent elements.
